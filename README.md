# Bootstrap Flower Shop Login Form

This repository contains a login form for a flower shop, built using Bootstrap framework version 5.3.3. The project showcases the combination of HTML and CSS to create a responsive and aesthetically pleasing login form.

## Cloning the Project

To clone the project, use the following command:

```bash
git clone "https://github.com/mihaiapostol14/Bootstrap-Flower-Shop-Login-Form.git"
```

## Project Structure

The project structure is simple and primarily consists of HTML and CSS files.

## Project Composition

The language composition of this repository is as follows:
- **HTML**: 62%
- **CSS**: 38%

## Features

- Responsive design using Bootstrap.
- Clean and modern UI.
- Easy to integrate into any web project.

## Setup Instructions

1. Clone the repository using the command above.
2. Open the project in your preferred code editor.
3. Open `index.html` in a web browser to view the login form.
